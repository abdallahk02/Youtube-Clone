# YouTube Clone

A modern YouTube clone built with React 19 and Vite, replicating YouTube's core UI and functionality. This project demonstrates component-based UI development, responsive design, and integration with the YouTube Data API.

---

## Features
- **Video Feed:** Displays popular videos from YouTube API with thumbnails, titles, view counts, and upload timestamps
- **Search Interface:** Search box in navbar for video discovery
- **Responsive Navigation:** Sidebar toggle functionality with menu icon
- **Video Categories:** Filter videos by category
- **User Profile Section:** Notification, upload, and profile icons in navbar
- **Video Player:** Dedicated page for playing individual videos
- **Recommended Videos:** Shows related videos on the video detail page

### Main Components
- **Navbar:** Top navigation bar with logo, search, upload, notifications, and profile icons
- **Sidebar:** Collapsible navigation menu
- **Feed:** Grid display of video cards fetched from YouTube API
- **PlayVideo:** Video player component for individual videos
- **Recommended:** Related/recommended videos section
- **Pages:** Home (main feed), Video (video detail)

---

## Project Structure
- `src/` - Main source code (components, pages, utilities)
- `public/` - Static assets
- `assets/` - Images and icons
- `index.html` - Main HTML file
- `package.json` - Project metadata and scripts
